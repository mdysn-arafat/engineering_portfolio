Finite Element Method (FEM) Projects

This repository contains two finite element analysis (FEM) studies carried out for structural verification and classification society review purposes.

1. Direct Strength Calculation of Deck Structure under Panama Chock Load (BV Approved)
This project involves the direct strength assessment of a deck structure subjected to Panama chock loading conditions. The analysis was performed using the Finite Element Method to evaluate stress distribution and structural response under design loads. The results were reviewed and approved by Bureau Veritas (BV) as part of the class approval process.

2. Crane Foundation Scantling Validation (RINA Class Review)
This project focuses on the validation of the scantling design for a crane foundation structure. A detailed FEM analysis was conducted to verify structural adequacy under operational and design load cases. The study was reviewed by RINA classification society, successfully addressing and closing A1 remarks during the approval process.
